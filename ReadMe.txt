Простое алгоритмическое решение задачи:
Важно:
    1) Защит от дурака по минимуму
    2) по требуемым данным - файл должен быть упорядочен по метрам времени, если текущая метка времени меньше
    предыдущей - строка будет игнорироваться ("Метки времени монотонно возрастают.")
    3) если закз удален раньше чем появился - строка проигнорируется
    4) если цена отрицательная - строка проигнорируется
    5) тип записи не "Е" или "I" - строка проигонорируется
    6) лишнии поля в записи 4, 5, 6 и прочие будут проигнорированы, т.е. программа прочитает только первые три или
    четыре поля
Как это вот работает:
    запускаем без праметров, программа потребует ввода строки, если с праметрами ничего не просит и далее
    считает все что просили задании (максимальную взмешеную по времени, максималную текщую и выводит спиок открытых
    заказов),  - ничего вводить не надо.
